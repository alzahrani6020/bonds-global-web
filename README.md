# موقع بوندز (مسودة أولى)

هيكل صفحة واحدة يلهم **أسلوب المواقع الاحترافية للاستشارات** دون نسخ نصوص أو بيانات مواقع أخرى.

## الشعار المرجعي (`شعار بوندز.jpg`)

- ضع الملف **كما هو** في `assets/شعار بوندز.jpg` (بدون تغيير أبعاد المصدر أو ضغط يغيّر النسبة). راجع `assets/README.txt` لنص **OCR** والألوان الثابتة.
- مكوّن `.brand-logo` يستخدم **`object-fit: contain`** لمنع تشويه الشعار في أي مقاس عرض.

## نكمل من هنا (ترتيب مقترح)

1. نسخ **`شعار بوندز.jpg`** إلى `assets/`.
2. **رفع المجلد كاملاً** إلى استضافة `bonds-global.com` مع HTTPS.
3. **حل 403** إن ظهر للزوار.
4. **نموذج التواصل** — Formspree أو بريد الاستضافة.
5. **فيسبوك** — رابط الموقع الرسمي.

تمت إضافة **canonical** و**Open Graph** في `index.html` (صورة المشاركة تشير إلى نفس ملف JPG بعد الرفع).

افتح `index.html` في المتصفح، أو شغّل خادماً بسيطاً من المجلد:

```bash
npx --yes serve .
```

## ما الذي يجب تعديله قبل النشر؟

1. قسم **القيادة والفريق**: الاسم، السيرة، الشهادات الحقيقية فقط.
2. **الإحصائيات** (`data-target` داخل `.stat__num`): أرقام صادقة أو احذف القسم.
3. **نموذج التواصل**: اربط الإرسال ببريد، أو Formspree، أو استضافتك.
4. **المقالات**: حوّل الروابط إلى مقالات فعلية أو احذف القسم مؤقتاً.
5. ألوان واجهة الموقع مربوطة بمتغيرات **شعار بوندز.jpg** في `styles.css` (`--bonds-*`).

## حقوق

 المحتوى هنا مكتوب لصالح مشروعك. لا تنسخ نصوص المواقع الأخرى حرفياً.
